Battleship version 1.0 05/07/2017

USAGE NOTES
-----------
This is a program written for CS 2365 Project04. It emulates the classic Battleship board game.
This is a single player game in which the user plays against a computer opponent.
The user may guess at cells on the game board in an attempt to locate the enemy's hidden ships.
Correct guessed turn the cell red, and incorrect guessed turn the cell blue.
The first side to successfully sink all 14 of their opponent's ship cells wins.

Installation
------------
To run the program, simply run the Project04Jar executable in this folder.
